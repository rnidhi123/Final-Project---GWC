My Coding Experience

Scratch and Tynker

I first started coding with Scratch and Tynker (the ones where you drag and drop blocks to form the code) in 5th grade. In our free time, we got to code on Scratch. When we went to the computer lab, we would sometimes do Tynker's Hour of Code.

I did not find coding interesting in 5th grade. In fact, I found it a little boring. Why? I thought that dragging and dropping blocks was easy, and that I needed a bigger challenge. But at that age, I didn't know that actual coding was more complex, and did not think much of coding.

In one of my 6th grade classes, I got Tynker. I thought that maybe it deserved a second chance, and I'm glad I gave it one. I gained so much more knowledge in the world of code, and I was proud of my creations.

Java Classes

Now I had become really interested in code, and I hungered for more. One day, my mom asked me if I wanted to join Java classes, and I immediately said yes. I started created wonderful games using Java, which turned out to be a more complicated language than I previously thought. I enjoyed and still enjoy every single one of my classes.

Girls Who Code

By now, I loved code. I also got the opportunity to join my school's Girls Who Code club. That is where I created this website. I learned many languages here, including CSS, HTML, and Javascript. I really could understand coding by now, and knew how to create my own projects. I could have never loved coding more than I do now.
