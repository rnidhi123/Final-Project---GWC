My Coding Experience

Scratch and Tynker

I first started coding with Scratch and Tynker (the ones where you drag and drop blocks to form the code) in elementary school. In my free time, I used Scratch to animate characters through code. When I was in the computer lab, I would sometimes use Tynker's Hour of Code program and create mini-games.

I did not find coding interesting in 5th grade. In fact, I found it a little boring. Why? I didn't know how to use blocks to form code, so I had a hard time creating projects, so I found coding boring. But at that time, I didn't know that actual coding was more complex.

In 6th grade, one of my classes was Tynker. I thought that maybe it deserved a second chance, and it was actually really fun and interesting. I gained so much more knowledge about code, and I was proud of my creations.

Java Classes

Now I found coding enjoyable, and I wanted to learn more. One day, my mom asked me if I wanted to join Java classes, and I immediately agreed. I started created fun games using Java, which turned out to be a more complicated language than I previously thought. I always enjoy going to my Java classes.

Girls Who Code

By now, I loved coding. I also got the opportunity to become a member of my school's Girls Who Code club, which is where I created this website. I learned many languages here, including HTML, Javascript, and CSS. I could understand code very well by now, and knew how to create my own projects. Being in this club was a really fun and memorable experience.
