My Coding Experience

Scratch and Tynker

I first started coding with Scratch and Tynker (the ones where you drag and drop blocks to form the code) in elementary school. In my free time, I used Scratch to animate characters through code. When I was in the computer lab, I would sometimes use Tynker's Hour of Code program and create mini-games.

I did not find coding interesting in 5th grade. In fact, I found it a little boring. Why? I thought that dragging and dropping blocks was easy, and that I needed a bigger challenge. But at that time, I didn't know that actual coding was more complex, and did I not think much of coding.

In 6th grade, one of my classes was Tynker. I thought that maybe it deserved a second chance, and I found out that it really did. I gained so much more knowledge in the world of code, and I was proud of the projects that I made.

Java Classes

Now I had become really interested in code, and I hungered for more. One day, my mom asked me if I wanted to join Java classes, and I immediately agreed. I was extremely thrilled for my first class. I started created wonderful games using Java, which turned out to be a more complicated language than I previously thought. I always enjoy going to my Java classes.

Girls Who Code

By now, I loved code. I also got the opportunity to become a member of my school's Girls Who Code club, which is where I created this website. I learned many languages here, including CSS, HTML, and Javascript. I really could understand coding by now, and knew how to create my own projects. I could have never loved coding more than I do now.
